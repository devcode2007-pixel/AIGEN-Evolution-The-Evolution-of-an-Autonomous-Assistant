# AIGEN: The Evolution of an Autonomous Assistant 🧬

Welcome to the development journey of **AIGEN**. This project documents my path from learning basic Python to building a complex Multi-Agent system. 

---

## Project Roadmap (The Generations)

### Gen 1: The Foundation
* **Status:** Completed
* **Goal:** Establish basic LLM connection using Ollama/Qwen.
* **Key Features:** Chat interface, file system interaction, and weather updates.
* **Hardware:** Optimized for Intel i3 / 8GB RAM.

### Gen 2: Persistent Memory
* **Status:** Completed
* **Goal:** Solve the "AI Amnesia" problem.
* **Key Features:** **MySQL Database** integration for long-term memory and user context retention.
* **Real-world Use:** Data management for the `MTN Shop` (Laptop E-commerce project).

### Gen 3: Professional Multi-Agent System
* **Status:** Active Development 
* **Hardware:** Powered by **Dell Precision 7550 (i7-10850H / 16GB RAM)**.
* **Key Features:**
    * **Agent 0 (Checker):** Logic and syntax verification.
    * **Agent 1 (Reviewer):** Security auditing and code optimization.
    * **Agent 2 (Test Writer):** Automated Unit Testing.
    * **Security Layer:** Advanced regex filtering for safe command execution.

---

##  Technology Stack
* **Languages:** Python, SQL, (C++ in progress).
* **Database:** MySQL.
* **AI Engine:** Ollama (Qwen 2.5 Coder, Llama 3.1).
* **Environment:** Visual Studio Code & XAMPP.
